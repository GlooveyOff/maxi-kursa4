# EduPlatform Backend

Backend для образовательной платформы с курсами. Курсовая работа.

## Стек

- Python 3.11+
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT (для авторизации)

## Запуск

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

После запуска документация доступна по адресу http://127.0.0.1:8000/docs
